# Learning Notes

## 今天練習什麼

這個專案適合剛開始學前端的人。你會看到一個小型網頁專案通常如何分工：

- `index.html`：放畫面上的元素。
- `src/style.css`：讓畫面好看，也處理手機版。
- `src/app.js`：處理新增、刪除、篩選、保存資料。

## 可以觀察的程式重點

1. `addEventListener` 讓網頁可以回應使用者操作。
2. `todos` 陣列保存目前所有待辦事項。
3. `render()` 每次資料改變後重新畫畫面。
4. `localStorage` 讓重新整理頁面後資料還在。

## 新手要理解的觀念

### 資料是主角

畫面只是資料的呈現。這個專案裡真正重要的是 `todos` 陣列。新增、刪除、勾選完成，其實都是在改變這個陣列。

### render 是重新整理畫面

每次資料改變後，程式會呼叫 `render()`。這個函式會根據目前的 `todos` 和篩選條件重新產生清單。

### localStorage 是簡單資料庫

`localStorage` 可以把資料存在瀏覽器裡。它不適合放機密資料，但很適合這種個人練習用的小工具。
