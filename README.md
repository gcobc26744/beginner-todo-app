# Beginner Todo App

這是一個適合新手練習的待辦清單小專案。它只使用 HTML、CSS 和 JavaScript，不需要安裝任何套件，打開瀏覽器就可以使用。

## 專案目標

- 練習基本網頁結構：HTML
- 練習版面與互動狀態：CSS
- 練習表單、事件監聽、陣列資料處理：JavaScript
- 練習把資料存進瀏覽器的 `localStorage`

## 功能

- 新增待辦事項
- 標記完成或未完成
- 刪除單一待辦事項
- 篩選全部、未完成、已完成
- 顯示剩餘待辦數量
- 資料會保存在目前瀏覽器

## 資料夾說明

```text
beginner-todo-app/
├─ README.md              # 專案介紹、功能、使用方式
├─ index.html             # 網頁入口，負責畫面結構
├─ src/
│  ├─ style.css           # 所有視覺樣式
│  └─ app.js              # 待辦清單互動邏輯
└─ docs/
   └─ learning-notes.md   # 新手學習筆記與延伸練習
```

## 如何使用

1. 下載或 clone 這個專案。
2. 用瀏覽器打開 `index.html`。
3. 在輸入框輸入待辦事項，按下 `Add`。

## 新手可以從這裡練習

- 把按鈕文字改成中文
- 增加「全部清除」功能
- 讓待辦事項可以編輯
- 加上到期日
- 改成深色模式

## 學到的重點

這個專案把網頁互動拆成三個部分：

- HTML 負責內容與結構。
- CSS 負責畫面與狀態樣式。
- JavaScript 負責使用者操作與資料更新。

當你可以看懂這三個檔案如何合作，就已經掌握很多前端專案的基本骨架。
